CO2 Calculator

Build a prototype for an online calculation tool for web CO2 emissions

The project was to create a tool aimed at managers and other decision makers to help create awareness. Together with an interactive interface for the user to be able to preview future improvements.

User Stories

As I user I want to be able to insert a URL so I can test if my website is considered green or not.

As I user I want to be able to select my industry so, potentially, I can make benchmark with other industries.

In this first step, the user enters a valid URL and selects the industry. As this was a demo, the data is not stored in any database so the industries are not saved. Then the URL is fetched from the form and run through two APIs, pageSpeed and Website Carbon

Because it takes a while, a loader is showed to the users to give feedback. After that, all the information displayed on the website is chaged according to the results of both APIs.

And some more User Stories

As I user I want to be able to make parameter changes so I can see how that affects my score.

As I user I want to be able to have a unique URL so I can share my result with a colleague.

In this case, the users can interact with different optioons presented which would have a positive impact. The idea behind it is that the agency can start the conversation with the C level to demonstrate what they would be able to achieve.

The new score according to the selected opportunities is just a demo. The dashboard data also changes accordingly to have a new visual. Lastly, the sharable link is also a prototype since no database is used.
