Redesign of the website

The first step was to make sure the website had enough contrast and was easily navegable. After some testing, we could that the user got lost and was confused about what the message was. Therefore, we changed the color palette and redesigned the UI.

Good typography not only makes the user read, but also feel. It allows the user to focus on the content and it affects how the information is erceived. It sets the personality of the website and affects the user's overall experience.

Original website lacking color contrast and CTA. Nav bar could use margin, so that all items are accessible.

Redesigned website with a video showing different people wearing different outfits, two CTA and increased contrast with the new color palette in place.

Connecting to a Database and making the website dynamic

We created filters on the top part, so that the user can easily access what they are searching for. On the side, category filter which after doing research, found out that was the main expectation from the users.

When clicking on the different categories, the list content changes accoordingly. We did this by fetching it from the url and sending a response of item matching the category. Since we didn't have access to the entire data base, we implemented fake products to test it out.
