Hogwarts Database

Build a system to handle the student list of Hogwarts College

The project was to create an interface for the administrators of the school to handle student lists. Some of the requirements were to be able to sort and filter. Also the interface must provide a "popup" window with detailed information about each student, including photo and house-crest and colors.

Creating the interface

The first step was to clean the data provided, spliting a full name into first name, alias and last name. The list provided is intended for administrators to get a quick overview of the students in the current year, and sort, filter, and search for certain properties.

The administrator is able to sort on First Name, Last Name and House, as well as by "Prefect state".

See the Pen Hogwarts List by Paula (@paulafiaschi) on CodePen.

Prefects, blood-status and Inquisitorial Squad Algorithms

Prefects:

The user must be able to make any student a prefect, and also revoke the prefect-status at any time. However, there can't be more than two prefects from each house. Therefore, the user must choos which student to remove from the list in order to be able to appoint the new one as Prefect.

Inq. Squad:

The user must be able to appoint students to the inquisitorial squad, and remove them again. Any number of students can be appointed, but only pure-blood or students from Slytherin (should any non-pure-blood be in that house).

Blood Status:

This is an indication of whether the student is from a pure wizarding family, or from a half-wizard, half-muggle family, or just plain muggle. There is a list of all known pure-blooded wizarding families and a list of the known half-bloods.

Hack the system

After many requests from the administrators, I got tired and decided to hack the system. First I infiltrate myself in the school, so I "inject" myself in the list of students, and make sure that I can't be expelled by the administrator.
Additionally I break the inquisitorial squad by removing students, shortly after the administrator has added them. And finally I mess up with the pure-blood algorithm, so it is rather random whether a student is shown as pure-, half-, or muggle-blood.
